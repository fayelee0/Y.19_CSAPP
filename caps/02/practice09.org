** A

   | 颜色   | 补色   |
   |--------+--------|
   | 黑色   | 白色   |
   | 蓝色   | 黄色   |
   | 绿色   | 红紫色 |
   | 蓝绿色 | 红色   |
   | 红色   | 蓝绿色 |
   | 红紫色 | 绿色   |
   | 黄色   | 蓝色   |
   | 白色   | 黑色   |


** B

   蓝色 | 绿色   = 001 | 010 = 011 = 蓝绿色

   黄色 & 蓝绿色 = 110 & 011 = 010 = 绿色

   红色 ^ 红紫色 = 100 ^ 101 = 001 = 蓝色
